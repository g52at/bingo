## 概要
ビンゴのルーレットのアプリです。
動作環境を選ばないようJQueryなどを用いずhtmlファイルのみで動作する設計です。

## 使用方法

htmlファイルをダウンロードしてクリックするだけで動作します。本当にシンプルです。
何か不具合が生じたり最初から動かしたい場合はブラウザをリロードしてください。
アンインストールはそのままゴミ箱に捨ててください。
